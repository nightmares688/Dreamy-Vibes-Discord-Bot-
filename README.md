# 🎵 Discord Music Bot

A simple Discord music bot built using `discord.py` that plays audio from YouTube.

---

## 🚀 Features

* 🎶 Play music from YouTube URLs
* ⏸ Pause / Resume songs
* ⏹ Stop playback
* 🔊 Join & leave voice channels

---

## 🛠️ Requirements

* Python 3.11+
* FFmpeg
* Discord Bot Token

---

## 📦 Installation

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
pip install -r requirements.txt
```

---

## ⚙️ Setup

1. Create a bot on Discord Developer Portal
2. Copy your bot token
3. Set environment variable:

```bash
TOKEN=your_bot_token
```

---

## ▶️ Run the bot

```bash
python music_bot.py
```

---

## ☁️ Deploy (Railway)

1. Upload project to GitHub
2. Connect repo to Railway
3. Add environment variable:

```
TOKEN=your_bot_token
```

4. Deploy 🚀

---

## 🎮 Commands

* `/join` → Join voice channel
* `/leave` → Leave voice channel
* `/play <url>` → Play music
* `/pause` → Pause
* `/resume` → Resume
* `/stop` → Stop

---

## ⚠️ Notes

* Make sure FFmpeg is installed
* Voice features require `PyNaCl`
* Free hosting may disconnect sometimes
